Employee Task

With the help of this module you can see number of tasks assigned to the employee.

Features

Allow User to See Number Of Tasks.

User can see the number of tasks assigned to the employee.

Stages Assigned to The Employee.

The list of tasks group by stages assigned to the employee.
Demo Request Video Tutorial

After installing this module, you can check the following access right to allow the particular user to see the number of tasks assigned to the employee in kanban and form view.

Following is the employee kanban view ,It is showing the number of tasks assigned to the employee.

Following is the form view of the employee. In the upper right side, It displays the number of tasks assigned to the employee.
You must have to select Related User

By Clicking on the 'Tasks' button, It will open the list of tasks group by stages assigned to this employee.
