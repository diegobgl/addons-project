Encrypted Vault

Overview

This module uses encrypted fields to store sensitive credentials, and related information.
Refer to Encrypted Field module to learn more about encryption method.

Everything except name, username and uri fields is encrypted. Custom field can be visible text, hidden text, or boolean.

By default password is getting generated, you can specify an existing password or generate new password at any moment. Password generator can be configured to generate password of desired length and include/exclude letters, numbers and special character.

All changes in Vault are kept in history which is also encrypted. You can easily restore any previous state.
