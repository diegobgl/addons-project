==============
Scrummer Scrum
==============

This module extends Scrummer with components needed for managing projects
with Scrum methodology. It creates backlog views, sprints, etc.

Usage
=====
Enable Use Agile on projects and choose Scrum as type. Add those projects
to Scrum team and open Scrummer. In Scrummer, you can create tasks and
sprints from Floating action button (Bottom right corner).
Drag tasks from Backlog to Sprint, start Sprint and open Active Sprint view
where you can drag and drop tasks as progress is made.
