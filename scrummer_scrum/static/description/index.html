Scrummer Scrum

Overview

This module extends Scrummer with components needed for managing projects with Scrum methodology. It creates backlog views, sprints, etc.

Usage

Enable Use Agile on projects and choose Scrum as type. Add those projects to Scrum team and open Scrummer. In Scrummer, you can create tasks...

...and sprints from Floating action button (Bottom right corner). Drag tasks from Backlog to Sprint and start Sprint...

...and start Sprint. In Active sprint view, you can see columns based on agile board, and you can move tasks according to mrojects workflow.
