Odoo Task Screenshot

First of all you should install this extension.

Features:

  • ☑   Create Task with screenshots in one click;
  • ☑   Attach screenshots to the task (if open project.task form view);
  • ☑   Redirect where screenshot was made, share;
  • ☑   If you clone database you can see where screenshot was made at you domain;
  • ☑   You can attach screenshot like solution of a task.

Create a task in any place in the Odoo system

If you open another one you can update it

Open link where screenshot was made; replace a domain; share
