Web Gantt View for Project Task in Odoo Enterprise

This apps helps user to view web gantt view for project task in quarter day and half day.

Half Day Quarter Day and View of Task

User can see task by half day and quarter Day filter on gantt view.

Date Details on Gantt view in Task and Planning Date on Work Order

User can see project task duration from assign date to deadline and Planning Date on Work Order on gantt view.

Days Duration Between Start Date to End Date on Gantt view in Leave

User can see leaves from start date to end date on gantt view.
